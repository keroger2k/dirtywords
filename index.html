Dirty Word Configuration Tool (v0.03)

Use this tool to create a Fortigate Firewall Configuration File for Dirty Word Search.

  1. INPUT: Comma Seperated Values (CSV) File containing a list of dirty words.
    e.g. "dirty1,dirty2,dirty3,dirty4"
    words containing '//' will be escaped with '\/\/'
  2. Enter the starting integer from the existing configuration.
    For example if your last rule is 100, enter 101.
  3. Enter the Virtual Domain to create dirty word list.
    Every VDOM has a unique dirty word list and must be maintained individually
  4. Copy and Paste entire list of dirty words into text box below
  5. Click Create to generate a configuration that can be utilized by the fortigate firewall to identify dirty words in XMPP traffic.
  6. Copy output results into a text file and import into Fortigate CLI via the console port, ssh, or the web based utility.
